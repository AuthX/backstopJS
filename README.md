# backstopJS

Quick Start
Anywhere in directory, to run references use the command:

npm run reference -- --configPath=PathToJsFile.js

example:
npm run reference -- --configPath=Projects/Unit4/backstop.js


To run tests:

npm run test -- --configPath=PathToJsFile.js
example:
npm run reference -- --configPath=Projects/Unit4/backstop.js
